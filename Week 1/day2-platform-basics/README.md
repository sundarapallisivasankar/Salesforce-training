# Salesforce Platform Basics

## What is Salesforce Platform?

Salesforce is a cloud-based CRM platform that helps businesses manage customer relationships, sales, services, and automation. It provides tools for both administrators and developers to build business applications.

---

# App

An App in Salesforce is a collection of objects, tabs, and functionalities designed for a specific business process.

Example:
- Sales App
- Service App

---

# Object

An Object is a database table used to store data in Salesforce.

Examples:
- Account
- Contact
- Opportunity

Objects contain:
- Fields
- Records

---

# Tab

A Tab is a user interface component used to access objects and features.

Examples:
- Accounts Tab
- Contacts Tab

---

# Configuration vs Coding

## Configuration (No Code)

Used for simple business logic using Salesforce tools.

Examples:
- Validation Rules
- Flow Automation

## Coding (Apex)

Used for advanced customization and complex logic.

Examples:
- API Integration
- Custom Automation

# Multi-Tenant Architecture

Salesforce uses multi-tenant architecture where multiple organizations share the same infrastructure securely.

Benefits:
- Scalability
- Security
- Low maintenance
- Automatic updates
